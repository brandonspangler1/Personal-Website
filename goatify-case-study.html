Goatify

--

Online Music Player

Project Goals

1. Stores songs in database with associated information

- Song name, artist, album, release year, genre

2. Allow Users to create an account

3. Allow Users to create multiple playlists and save those in the database

4. Must be useable on any device with a web browser

5. Allow Users to Search for songs with any of the following

- Song name, artist name, album name

6. Allow Users to filter search results by

- Release year, genre

7. A new User must be forced to create a playlist immediately

8. The music player must have the following features

- Play, pause, skip, and previous song

- Play current song/playlist in a loop

- Not download all the songs to the User's device
